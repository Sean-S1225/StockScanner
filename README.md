# Stock Scanner

A Python-based stock market screening and visualization tool designed for swing trading.

Swing trading, unlike day trading, focuses on identifying stocks that may be near the end of a broader downswing, with the goal of entering positions before a potential rebound. This project began as a sandbox for exploring financial data and technical indicators, and is gradually evolving into a more flexible framework for market screening, chart analysis, and strategy development.

## Current Features

- Screen stocks using a relatively sophisticated, fixed set of technical criteria
- Plot historical price data as candlestick charts across multiple timescales, including weekly and daily views
- Support a growing set of technical indicators, including:
  - Exponential Moving Averages (EMA)
  - MACD
  - VMC
  - Fibonacci levels

## Planned Features

- A backtesting engine with customizable:
  - screening criteria
  - buying strategies
  - selling strategies
- More flexible indicator configuration and chart customization
- Improved ranking and comparison of screened stocks
- Better modularity for experimenting with new trading ideas

## Project Goals

This project is intended to serve two purposes:

1. provide a practical tool for identifying swing trading opportunities
2. act as a personal software project for learning more about market data, technical analysis, and quantitative strategy design

## Status

This project is actively under development and the API, features, and screening logic may change over time.
